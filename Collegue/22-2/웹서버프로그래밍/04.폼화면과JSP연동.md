# select, textarea 태그
### select 태그
- 폼 태그 안에서 사용
- option 태그로 선택할 값들을 지정

```html
<select name="number">
	<option value="choice">선택</option>
	<option value="1">1</option>
	<option value="2">2</option>
	<option value="3">3</option>
</select>
```

### textarea 태그
- 여러줄의 텍스트를 입력할 수 있는 태그
```html
<textarea cols="너비" rows="높이">
</textarea>
```
