# 아두이노 IDE
> 학습내용
- 아두이노 프로그램 실행

### 아두이노IDE
- java 기반으로 OS간 이식성 우수

### 아두이노 SW 장점 - 추상화된 함수
- 마이크로 컨트롤러 마다 다른 명령어를 추상화 해서 동일한 코드로 작동하게 해준다

### 아두이노와 PC 연결
- PC와 아두이노 보드가 연결되면, 자동으로 드라이버 설치

### 스케치의 구조
- C/C++을 기반으로 한다
- main함수는 내부적으로 숨겨져 있어서 구현x
- 2개의 기본 함수로 구성
	- setup : 초기화 함수로 시작시 한번만 실행
	- loop : 반복 실행 함수로 메인/이벤트 루프
