# 운영체제 개념
## 운영체제 동작
- 컴퓨터 부팅 과정
    1. ROM의 Bootstrap에 저장된 초기 프로그램 수행
    2. OS를 메모리에 로드 자원 초기화, 커널 생성
## 운영체제 개념
- 하드웨어의 자원을 효율적으로 관리,운영하는 소프트웨어
    - 프로세스, 메모리, 저장장치, 주변장치, 입출력 장치 관리
    - 사용자와 하드웨어 사이의 중계 담당
        - 사용자 <-> 응용 프로그램 <-*시스템 함수(System Call)*-> 운영체제 <-*커널, 장치드라이버*-> 하드웨어
        - 대부분의 커널 함수는 시스템 콜 함수를 이용한다
