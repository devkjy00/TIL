# SQL실습환경구축
> 학습내용
- 오라클 데이터베이스 11g XE 특징
- SQL*Plus, SQL Developer

## 오라클 데이터베이스 11g XE
- RDBMS, ORDBMS : 관계형 모델 및 객체 관계형 모델 지워
- XE(Express Edition) : 엔트리 급의 가벼운 데이터 베이스
    - 개발, 설치, 배포 자유로움, 성능제한

- SQL*Plus
    - CLI 환경에서 동작, 기본적으로 설치된다

- SQL Developer
    - GUI 환경에서 동작, 다운로드후 압출풀고 실행
    - 데이터 베이스 연결, 17p
