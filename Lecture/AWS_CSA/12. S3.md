### S3 보안 
- User-Based 
	- IAM Policies 
- Resource-Based 
	- Bucket Policies : account를 허용하는 방법
		- JSON으로 작성
	- Object Access Control List (ACL)  : 가장 일반적인 방법
	- Bucket Access Control List (ACL)  

- Encryption : 암호키로 암호화 할 수 있다

- use case
	- 사용자가 직접 접근 : IAM permissions
	- EC2 에서 접근 : IAM Roles
	- 다른 계정의 사용자가 접근 : Bucket policy
