### AWS 글로벌 인프라
 - 리전(Regions)
	 - 리전을 선택하지 않는 글로벌 서비스도 있다(IAM...)
	 - ***재해복구(DR)설계  : 복수의 리전에 서비스 배치***
 - 가용 영역(Availability Zones)
	 - 하나 이사의 개별 데이터 센터(보통 3~4개의 가용영역)
	 - 물리적으로 떨어져 있다
	 - ***고가용성 설계 : Multi-AZ 가능***
 - 데이터 센터(Data Centers)
 - 엣지 로케이션(Edge Location)
	 - 서버, 컨텐츠 캐싱용도
	 - 리전이 먼 경우 가까운 엣지 로케이션에 캐싱해서 사용
