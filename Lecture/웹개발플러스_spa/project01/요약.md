## 설정
- mac mongoDB 설치
    - brew tap mongodb/brew -> 저장소 추가
    - brew install mongodb-community -> 설치
    - brew services start mongodb-community -> 실행
    - localhost:27017으로 접속했을 때 영문 출력되면 성공
