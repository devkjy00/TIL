# JQuery 
- 개요
    - 자유 오픈 소프트웨어로 한 개의 JavaScript 파일로 존재한다
    - CSS 셀렉터 기반 DOM 탐색 및 수정
    - AJAX
    - JSON parsing
    - 이벤트, 특수효과 및 애니메이션

- 두가지 상호 작용 스타일
    - $ : JQuery 오브젝트의 팩토리 메소드 사용
        - 주로 DOM노드들을 조작하는 함수
        - JQuery 객체를 반환 연계해서 함수를 호출할 수 있다
    - $. : 이 함수들은 JQuery 오브젝트 그 자체와 연동되지는 않는다
        - each, ajax ...
