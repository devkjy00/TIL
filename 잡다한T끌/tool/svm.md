# Subversion(svn)
- 서버를 사내 서버에 설치해서 사용한다
- 클라이언트는 Command Line, Tortoise SVN(클라이언트), Subversive SVN Tema Provider(이클립스 플러그인)으로 사용된다

### 용어
- Revision : 0부터 1씩 증가해서 id로 활용, 가장 최근의 리비전을 HEAD Revision이라고 한다
- Working Copy : 체크아웃을 통해 내려받은 복사본
- Add : git add
- Update : git pull
- Commit : git commit
- trnuk : branch main
