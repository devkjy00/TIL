## 분할 정복
#### 사용 포인트
- 이진 탐색 할 수 있는 자료구조 처리

#### 예시
```py
def divide(arr: list, left: int, right: int) -> int:
    if (left == right):
        return arr[left]

    mid = left + (right - left) / 2
    left_result = divide(arr, left, mid)
    right_result = divide(arr, mid + 1, right)

    return do_something(left_result, right_result)
```
